# Heart Disease Prediction using Machine Learning

## Overview
This project predicts whether a patient is likely to have heart disease using Machine Learning algorithms.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Algorithms
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Accuracy Comparison

## Dataset
UCI Heart Disease Dataset

## Results
The Random Forest model achieved the highest accuracy among the trained models.
